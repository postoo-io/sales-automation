## 단계 2 — 견적 계산

**입력 항목**

| # | 항목 | 단위 | 수량 | 단가 | 소계 |
|---|------|------|---:|---:|---:|
| 1 | AI 어시스턴트 도입 컨설팅 | M/M | 5 | 5,000,000 | 25,000,000 |
| 2 | 데이터 통합 PoC 개발 | M/M | 1.5 | 6,000,000 | 9,000,000 |
| 3 | 클라우드 인프라 1개월 | 식 | 1 | 1,500,000 | 1,500,000 |

**계산 과정**

```
소계 합계   = 25,000,000 + 9,000,000 + 1,500,000 = 35,500,000원
할인액      = 35,500,000 × 0% = 0원
과세표준    = 35,500,000원
부가세      = 35,500,000 × 10% = 3,550,000원  (vatType=E, 별도)
합계        = 35,500,000 + 3,550,000 = 39,050,000원
```

## 발행 후 실행 예시 (사용자가 "발행" 선택 시)

```bash
# 1. 의뢰 예상 견적 금액 갱신
PATCH /v1/inquiry/1234
{"estimate": "39050000"}

# 2. 견적 발행 메모 기록 (마진율·원가 미포함)
POST /v1/inquiry/1234/text_history
{"content": "견적 v1 발행 — 합계 39,050,000원 (부가세 별도 35,500,000원), 유효 2026-06-05, 할인 없음. 항목: AI 컨설팅 5M/M, 데이터 PoC 1.5M/M, 클라우드 인프라 1식."}

# 3. 신규 단가 라이브러리 등록 (클라우드 인프라)
POST /v1/estimate/item
{"title": "클라우드 인프라 구성 (1개월)", "unitCost": "1500000", "unit": "식"}

# 4. Drive 저장
# → /영업/2026/견적서/견적서_노바테크_20260522_v1.md
```

**저장 완료 확인**:

✅ 신규 단가 "클라우드 인프라 구성" 라이브러리 등록 완료
