## 단계 6 — 발송 후 의뢰 히스토리 기록 예시

**사용자가 "발송"을 선택했다고 가정:**

```bash
# Gmail MCP로 메일 발송 완료
# → 발송 완료 후 Pluuug 의뢰 히스토리에 자동 기록

POST /v1/inquiry/1234/text_history
```

```json
{
  "content": "이메일 발송 — 2026-05-22 10:35\n제목: [(주)노바테크] AI 어시스턴트 도입 PoC 견적서 송부 드립니다\n수신: hong@novatech.co.kr (CC: park@novatech.co.kr)\n첨부: 견적서_노바테크_20260522_v1.md\n템플릿: 30-견적송부"
}
```

발송 완료 확인:

```
✅ 이메일 발송 완료 (2026-05-22 10:35)
✅ Pluuug 의뢰 #1234 히스토리 기록 완료
```
